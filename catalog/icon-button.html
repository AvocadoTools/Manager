Styles

None

Slots

None

Properties

  • data (object): Arbitrary data storage

Attributes

  • concealed (boolean): Do not show, but leave in DOM flow
  • disabled (boolean): Disable the button element
  • filled (boolean): Fill variation on icon font
  • hidden (boolean): Do not show in DOM
  • href (string): URL to open when button is clicked
  • kind (danger | ghost | secondary | tertiary): Button styling
  • name (string): Reference to icon in Google Fonts collection
  • size (2xl | md | sm | xl): Height and width of button
  • src (string): Source for image icon
  • target (string): Specify window to use when opening a new URL (href)
  • weight (integer, 100 - 700): Weight variation on icon font

Events

  • adc-focus: Button element receives focus
  • adc-blur: Button element loses focus

Methods

  • blur(): Programmatic removable of focus from button element
  • click(): Programmatic click on the button element
  • focus(): Programmatic focus on button element

Parts

  • button: Button element
  • font: Paragraph element used inside the AvocadoIcon (adc-icon)
  • icon: AvocadoIcon (adc-icon) element
  • image: Image element used inside the AvocadoIcon (adc-icon)

Dependencies

  • adc-icon
