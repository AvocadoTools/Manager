Select

<adc-select error="Error" helper="Helper" label="Label"></adc-select>

Styles

None

Slots

None

Properties

  • data (object): Arbitrary data storage
  • options (array): The options to display in the select

Attributes

  • concealed (boolean): Do not show, but leave in DOM flow
  • disabled (boolean): Disable the button element
  • error (string): Content to display in error label
  • helper (string): Content for the helper label
  • hidden (boolean): Do not show in DOM
  • invalid (boolean): Indicate an invalid state on the select field
  • label (string): Content for the main label
  • light (boolean): Light background on select field
  • name (string): Name of the field represented by this input
  • read-only (boolean): Keep input styling, but do not allow edits
  • value (string): Value of the select field

Events

  • adc-blur: Select element loses focus
  • adc-change: User has changed the selected option
  • adc-focus: Select element receives focus
  • adc-input: Select element has had value set

Methods

  • add( item, before ): Add an option to the select element
  • blur(): Programmatic removable of focus from button element
  • focus(): Programmatic focus on button element
  • item( index ): Get the item at the provided index
  • itemize( option ): Serialize an option element into an object
  • remove( index ): Remove the item at the provided index

Parts

  • button: Button element
  • label: AvocadoLabel (adc-label) element
  • p: Paragraph element used inside the AvocadoLabel (adc-label)

Dependencies

  • adc-icon
  • adc-label
  • adc-vbox
