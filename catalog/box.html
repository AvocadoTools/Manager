Box

Useful for trimming down size where both HBox and VBox might have to otherwise be used in the same component. Intended for internal use.

<adc-box direction="row" gap="30">
  <adc-icon name="star"></adc-icon>
  <adc-label text="Label"></adc-label>  
</adc-box>

Styles

None

Slots

  • default: Interchangeable flex box layout

Properties

  • data (object): Arbitrary data storage

Attributes

  • concealed (boolean): Do not show, but leave in DOM flow
  • direction (boolean): Direction of flex box layout
  • disabled (boolean): Tells containing element to disable related part
  • gap (integer): Distance between contained items
  • helper (boolean): Informs additional text content used by containing element part
  • hidden (boolean): Do not show in DOM
  • icon (string): Specifies an icon to use in containing element part
  • label (string): Informs primary text content used by containing element part

Events

None

Methods

None

Parts

None

Dependencies

None
