Input

<adc-input 
  clearable 
  error="Error" 
  helper="Helper" 
  label="Label" 
  placeholder="Placeholder">
  <adc-link label="Link"></adc-link>
</adc-input>

Styles

None

Slots

  • default: Content placed on top of input, aligned to the right. Useful for letter counter, etc.
  • prefix: Content leading the input field; still in border
  • suffix: Content trailing the input field; still in border

Properties

  • data (object): Arbitrary data storage

Attributes

  • clearable (boolean): Show a button to clear the input field
  • concealed (boolean): Do not show, but leave in DOM flow
  • disabled (boolean): Disable the input field; also changes appearance
  • error (string): Content to display in error label
  • helper (string): Content for the helper label
  • hidden (boolean): Do not show in DOM
  • invalid (boolean): Indicate an invalid state on the field
  • label (string): Content for the main label
  • light (boolean): Light background on input field
  • mode (string): Keyboard hint on mobile devices
  • name (string): Name of the field represented by this input
  • placeholder (string): Placeholder for input label
  • readonly (boolean): Keep input styling, but do not allow edits
  • revealable (boolean): Show reveal button when input type is password
  • type (string): The type of input field
  • value (string): Value of the input field

Events

  • adc-blur: Button element loses focus
  • adc-change: Change in input field value
  • adc-clear: Input field cleared
  • adc-enter: Enter key was pressed on the input field
  • adc-focus: Button element receives focus
  • adc-input: Content of input field changed

Methods

  • blur(): Programmatic removable of focus from button element
  • focus(): Programmatic focus on button element
  • select(): Select all the text in the input field
  • setSelectionRange(): Select a portion of the input field text
  • setRangeText(): Replace selected portion of input field text
  • valueAsFloat(): Get the input field value as a float
  • valueAsInt(): Get the input field value as an integer

Parts

  • clear: HTML button element that shows clear state for input field
  • clear-icon: HTML paragraph element for the clear state AvocadoIcon (adc-icon)
  • error: Error message AvocadoLabel (adc-label) for input field
  • error-p: HTML paragraph element for the error AvocadoLabel (adc-label)
  • field: HTML label element that wraps the input field
  • helper: Helper text AvocadoLabel (adc-label) for input field
  • helper-p: HTML paragraph element for the helper AvocadoLabel (adc-label)
  • input: The HTML input field
  • invalid: The AvocadoIcon (adc-icon) element that shows invalid state
  • invalid-icon: HTML paragraph element for the invalid AvocadoIcon (adc-icon)
  • label: Main AvocadoLabel (adc-label) for input field
  • label-p: HTML paragraph element for the label AvocadoLabel (adc-label)
  • reveal: HTML button element that allows toggle in password state
  • reveal-icon: HTML paragraph element for the password reveal AvocadoIcon (adc-icon)

Dependencies

  • adc-icon
  • adc-label
