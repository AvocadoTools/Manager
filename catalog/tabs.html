Styles

  • --tabs-background-color: #f4f4f4

Slots

  • default: Placed inside AvocadoStack (adc-stack) element as tab panel content

Properties

  • data (object): Arbitrary data storage

Attributes

  • concealed (boolean): Do not show, but leave in DOM flow
  • expandable (boolean): Show icon to allow for expanded state
  • expanded (boolean): Show tabs in expanded state
  • hidden (boolean): Do not show in DOM
  • light (boolean): Light background of AvocadoStack (adc-stack) element
  • selected-index (integer): Index representing the visible child
  • tab-renderer (string): Provide your own component to be used for tab controls

Events

  • adc-change: The selected tab has changed
  • adc-collapse: The state is no longer expanded
  • adc-expand: The state is now expanded

Methods

  • collapse(): Programmatically set the state of the tabs to collapsed
  • expand(): Programmatically set the state of the tabs to expanded
  • show( index ): Programmatically control the visible child and selected tab

Parts

  • header: Container for the displayed tabs and zoom button
  • icon: The AvocadoIcon (adc-icon) inside the AvocadoIconButton (adc-icon-button)
  • tabs: Container for the displayed tabs
  • views: The AvocadoStack (adc-stack) element that controls visibility
  • zoom: The AvocadoIconButton (adc-icon-button) for controlling expanded state
  • zoom-p: The paragraph element underlying the AvocadoIcon (adc-icon) icon font

Dependencies

  • adc-hbox
  • adc-icon-button
  • adc-stack
  • adc-tab
