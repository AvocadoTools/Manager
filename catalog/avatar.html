Styles

  • --avatar-background-color: #f4f4f4
  • --avatar-size: 61px
  • --avatar-label-color: #161616
  • --avatar-label-font-size: 20px

Slots

  • icon: Specify a placeholder; usually a graphic slement

Properties

  • data (object): Arbitrary data storage
  • labelFunction (function): Function to call to transform label content
  • src (object): File path to image, or encoded image data
  • value (object): Represents the collective information about the avatar

Attributes

  • compression (float): How to compress any image resizing
  • concealed (boolean): Do not show, but leave in DOM flow
  • constraint (integer): Maximum size for image resizing
  • disabled (boolean): Not available for selection or change
  • hidden (boolean): Do not show in DOM
  • format (string): Image format for encoding of resized image
  • label (string): Label content to use when no image is present
  • light (boolean):
  • readOnly (boolean): Similar to disabled, but does not alter appearance
  • shorten (boolean): Flag to shorten any label content provided

Events

None

Methods

  • clear(): Clear label and image content
  • initials(): Algorithm for name shortening

Parts

  • button: Button element encapsulating functionality
  • label: Label content; usually a name
  • p: Paragraph element used in AvocadoLabel (adc-label)

Dependencies

  • adc-label
