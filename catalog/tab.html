Tab



<adc-tab helper="Helper" icon="star" label="Label" selected></adc-tab>
<adc-tab label="Label"></adc-tab>    
<adc-tab disabled label="Label"></adc-tab>

Styles

None

Slots

None

Properties

  • data (object): Arbitrary data storage

Attributes

  • concealed (boolean): Do not show, but leave in DOM flow
  • disabled (boolean): Tab is in disabled state
  • helper (string): Content for helper label
  • hidden (boolean): Do not show in DOM
  • icon (string): Icon font name to be show inside tab
  • label (string): Content for main label
  • selected (boolean): Tabs is in selected state

Events

None

Methods

None

Parts

  • button: Button representing container for tab elements
  • box: AvocadoHBox (adc-hbox) line of content containing label and icon
  • font: Paragraph element inside AvocadoIcon (adc-icon)
  • icon: AvocadoIcon (adc-icon) element for icon display
  • helper: Secondary AvocadoLabel (adc-label) for helper content
  • helper-p: Paragraph element inside AvocadoLabel (adc-label)
  • label: The AvocadoLabel (adc-label) representing the main label
  • label-p: Paragraph element inside the main AvocadoLabel (adc-label)

Dependencies

  • adc-hbox
  • adc-icon-button
  • adc-label
